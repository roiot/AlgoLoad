# Презентация по AlgoView / AlgoLoad

Это LaTeX-проект презентации, посвящённой:

- системе **AlgoView** (3D-визуализация и интерактивный анализ информационных графов алгоритмов);
- интеграции и использованию в рамках платформы практикумов **AlgoLoad**.

## Состав папки

- `main.tex` — основной файл сборки презентации;
- `slide*.tex` — содержимое слайдов;
- `images/`, `animated/` — графические материалы;
- `*.sty`, `*.cls` — стили и класс Beamer;
- `*.zip` (в родительской папке `LaTeX/`) — резервная копия проекта.

## Статус

Материалы находятся в рабочем, развивающемся состоянии: структура и содержимое слайдов могут изменяться по мере подготовки презентации и диссертации.
