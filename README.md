# AlgoLoad

Репозиторий **AlgoLoad** объединяет материалы и код проекта по визуализации и анализу информационных графов алгоритмов.

## Текущая структура

- `frontend/` — фронтенд на Flutter *(планируется добавить)*.
- `backend/` — серверная часть на Python *(планируется добавить)*.
- `LaTeX/` — LaTeX-проекты с научными материалами (презентации, диссертация и др.).

## Текущий статус

Сейчас репозиторий находится на промежуточном этапе: в первую очередь ведётся работа над презентацией в `LaTeX`.
По мере развития проекта сюда будут добавлены программные компоненты (frontend, backend и другие элементы системы).
